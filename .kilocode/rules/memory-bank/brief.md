# Project Brief

## Project Name
Marketing Portfolio Web App (marketing.tvoje.info)

## Purpose
High-performance, accessible, bilingual portfolio website showcasing DevOps and AI expertise for the Czech and international markets.

## Goals
- Position as premier DevOps/AI service provider in Central Europe
- Achieve 95+ Lighthouse scores across all metrics
- Capture 10+ monthly contact form submissions
- Rank in Google.cz and Seznam.cz for Czech keywords
- Serve both Czech and English audiences through bilingual content

## Scope
**In scope:**
- Astro 5.0 + Tailwind CSS 4.0 website
- Core sections: Hero, About, Projects, Services, Contact, Testimonials
- Bilingual support (Czech + English)
- WCAG 2.2 AA accessibility compliance
- Privacy-focused analytics (Plausible)
- Formspree contact form integration
- Performance optimization (95+ Lighthouse)

**Out of scope:**
- E-commerce functionality
- User authentication
- Real-time chat support
- User-generated content
- Payment processing

## Target Audience
1. Tech Founders (40%) - Startup leaders seeking DevOps/AI partners
2. Marketing Agency CTOs (25%) - Agency technical decision-makers
3. SMB Owners (20%) - Local business owners seeking affordable solutions
4. Czech Business Clients (15%) - Local market with language requirements

## Success Criteria
- Lighthouse Performance ≥95, Accessibility ≥95, SEO ≥95
- LCP <2.0s, FID <100ms, CLS <0.1
- Organic traffic: 500+ monthly visits within 3 months
- Contact form submissions: 10+ per month
- Czech traffic: 30%+ of total visitors
- Bounce rate: <40%
