# Product Description

## Product Overview
A modern, performant portfolio website built with Astro 5.0 and Tailwind CSS 4.0, designed to showcase DevOps and AI expertise to Czech and international audiences through bilingual content delivery.

## User Experience

### Primary User Journey
1. Visitor arrives from search (Google/Google.cz/Seznam.cz) or referral
2. Hero section communicates value proposition and credentials immediately
3. Projects showcase demonstrates capabilities through case studies
4. Services section outlines offerings and engagement models
5. Contact form enables conversion with GDPR-compliant data collection

### Design Philosophy
- **Minimalist yet professional** - Clean lines, generous whitespace, typography-led hierarchy
- **Performance-first** - Zero-JS by default, island architecture for selective interactivity
- **Accessible by default** - WCAG 2.2 AA compliance as foundation, not afterthought
- **Bilingual seamless** - Language toggle with persistent preference, no page reload

### Key Differentiators
1. **Bilingual DevOps/AI Portfolio** - Unique positioning for Czech market
2. **Automation Showcases** - Interactive CI/CD demonstrations proving capabilities
3. **Documentation-First Approach** - Comprehensive case studies with metrics and architecture diagrams

## Features

### Core Features (MVP)
- Hero section with value proposition and credentials
- About/Bio section with career timeline and certifications
- Projects showcase with filtering and case study links
- Services section with pricing reference
- Contact form (Formspree) with GDPR consent
- Testimonials section with client social proof
- Dark mode toggle with system preference detection
- Fully responsive mobile-first design

### Technical Features
- Astro 5.0 island architecture
- Tailwind CSS 4.0 styling
- TypeScript throughout
- Bilingual content (CZ/EN) with i18n
- WCAG 2.2 AA accessibility
- 95+ Lighthouse scores target
- Plausible privacy-focused analytics
- Vercel edge deployment

### Content Features
- Case studies with problem-solution-results structure
- Quantifiable metrics (percentage improvements, cost savings)
- Technology stack documentation
- Client testimonials with verification
- Blog section for thought leadership (v1.1)
