"## Technical Architecture" 
"" 
"### Frontend Architecture" 
"- **Framework**: Astro 5.0 with island architecture" 
"- **Styling**: Tailwind CSS 4.0 with custom design system" 
"- **Language**: TypeScript 5.7 for type safety" 
"- **Build**: Vite-based build system with optimization" 
"" 
"### Content Management" 
"- **Structure**: MDX for content with component embedding" 
"- **Internationalization**: i18n library for bilingual content" 
"- **Assets**: Optimized image pipeline with WebP/AVIF support" 
"- **Data**: JSON/YAML for structured data and metadata" 
"" 
"### Integration Points" 
"- **Forms**: Formspree for contact form handling" 
"- **Analytics**: Plausible for privacy-focused analytics" 
"- **Search**: Pagefind for static site search" 
"- **Deployment**: Vercel for hosting and CI/CD"
