# Designer Skill - Kilo Code

## Overview

Visual design implementation for 2026 modern websites. Google-inspired color palette with surprise elements.

## Usage

```
Use this skill when:
- Setting up design system
- Creating components
- Adding animations
- Ensuring accessibility
```

## Design System

```css

--secondary: #ea4335; /* Red */
--accent: #fbbc05; /* Yellow */

```

- Gradient mesh backgrounds
- Glassmorphism cards
- Micro-interactions
- Scroll animations
