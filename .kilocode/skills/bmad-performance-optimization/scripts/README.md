## Adding Scripts

When adding new scripts:
1. Create a descriptive filename with appropriate extension (.js, .py, .sh, etc.)
2. Document the script's purpose and usage in this README
3. Update the skill's REFERENCE.md if applicable
4. Ensure scripts are executable and version-controlled
5. Include error handling and logging

## Script Guidelines

- Use clear, descriptive variable names
- Include comments explaining complex logic
- Handle errors gracefully
- Log important events and metrics
- Make scripts idempotent when possible
