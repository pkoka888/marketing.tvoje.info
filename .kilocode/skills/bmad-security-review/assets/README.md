# Assets

This directory contains supporting resources for the Security Review skill.

## Contents

- **Templates**: Reusable templates for threat modeling, risk assessment, and remediation planning
- **Reference Materials**: Security standards, checklists, and best practices
- **Documentation**: Architecture diagrams, data flow documents, and compliance matrices

## Usage

Assets in this directory are referenced from the main skill files:
- SKILL.md - Skill definition and activation criteria
- CHECKLIST.md - Quality assurance checklist
- REFERENCE.md - Quick-access resources
- WORKFLOW.md - Step-by-step workflow

## File Types

- `.md.template` - Template files for security artifacts
- `.md` - Reference documentation and guides
- `.drawio` / `.png` - Architecture and threat model diagrams
- `.csv` - Data exports and tracking spreadsheets

## Notes

- Assets are optional and can be customized per project needs
- Templates should be copied and filled in, not edited directly
- Keep assets version-controlled alongside the skill
