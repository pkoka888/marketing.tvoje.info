# UX Design Assets

This directory contains templates and reference materials for UX design work.

## Contents

### Templates
- `user-flows-template.md.template` - Template for documenting user flows and journeys
- `wireframes-template.md.template` - Template for wireframe documentation
- `design-system-template.md.template` - Template for design system specifications

## Usage

These templates are used during the UX design workflow to create consistent, well-documented design artifacts. Each template includes:

- Structure guidelines
- Required sections
- Example content
- Best practices

## Notes

- All templates use `.template` extension to distinguish them from final deliverables
- Templates should be copied and customized for each project
- Final deliverables should be saved without the `.template` extension
