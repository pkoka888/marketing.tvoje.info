# Development Execution Scripts

This directory contains utility scripts for the Development Execution skill.

## Purpose

Scripts in this directory provide automation and helper functions for:
- Running tests
- Generating implementation notes
- Validating story files
- Checking code quality

## Usage

Scripts should be executable and follow these conventions:
- Use shebang line (`#!/usr/bin/env python3` or `#!/bin/bash`)
- Include usage comments
- Handle errors gracefully
- Provide clear output
