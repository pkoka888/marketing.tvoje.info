# Custom Modes

This directory contains project-specific custom modes for Kilo Code.

## Available Modes

Reference global modes in `.kilocode/` for configuration.

## Creating New Modes

See `.kilocode/custom_modes.yaml` in the global configuration.

## Prompt Consultant Mode

The `prompt` mode is designed for prompt engineering and optimization tasks. It provides specialized capabilities for:

- Analyzing and improving existing prompts
- Creating new prompts for specific use cases
- Prompt testing and validation
- Prompt versioning and management

To use the prompt consultant mode, switch to it using the mode selector in Kilo Code.
