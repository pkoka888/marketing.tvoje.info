// turbo-all

## Steps

1. Install dependencies if `node_modules/` doesn't exist:
```bash
npm install
```

2. Start the Astro dev server:
```bash
npm run dev
```

3. The server will be available at `http://localhost:4321`

## Notes
- Hot Module Replacement (HMR) is enabled by default
- Changes to `.astro`, `.ts`, `.css` files trigger instant refresh
- Stop with `Ctrl+C`
