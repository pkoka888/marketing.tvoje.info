# Active BMAD Workflows

This document lists the active BMAD (Business-Minded Agile Development) workflows available in this project.

## Workflow Categories

The following workflow categories are available in `_bmad/bmm/workflows`:

- **1-analysis**: Workflows for initial project analysis and research.
- **2-plan-workflows**: Planning and strategy workflows.
- **3-solutioning**: Architecture and solution design workflows.
- **4-implementation**: Implementation and coding workflows.
- **bmad-quick-flow**: Rapid development workflows.
- **document-project**: Documentation generation workflows.
- **generate-project-context**: Context gathering workflows.
- **qa**: Quality assurance and testing workflows.

## Usage

To use a workflow, navigate to the respective directory in `_bmad/bmm/workflows` and follow the instructions in the template files.
